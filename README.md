# **Evoteh - Online Product Catalog for Snowplows**

**Live Demo**: [evoteh.rs](https://evoteh.rs)

A website created in **PHP** for listing **snowplows**. The platform offers a simple interface for browsing products, viewing detailed specifications, and gathering information.

## **About**

A fully responsive website 'Evoteh' was created to highlight and offer comprehensive information about **snowplows**. The system, created with **PHP**, **JavaScript**, **Bootstrap**, and a database in **Microsoft SQL Server**, makes it simple for users to browse and explore products. To help consumers learn more about the equipment each product listing includes images, specifications and descriptions.

---

## **Features**

- Product browsing
- Secure backend powered by **PHP** and **MS SQL**

---

## **Tech Stack**

- **Backend**: PHP
- **Frontend**: HTML, CSS, Bootstrap, JavaScript
- **Database**: Microsoft SQL Server

## **Installation**

1. **Clone the repository**:
    ```bash
    git clone https://github.com/StefanSirkovic/evoteh-php.git
    cd evoteh-php
    ```
    
2. **Set up the environment**:
    - Configure your **PHP server** (e.g., XAMPP, WAMP) to point to the project directory.
      
3. **Start the local server**:
    - Open your preferred local server (e.g., XAMPP) and access the application:
      ```
      http://localhost/evoteh-php
      ```
